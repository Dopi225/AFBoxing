2. Accès à l'administration

1 Ouvrez votre navigateur (Chrome, Firefox, Edge, Safari)
2 Allez à l'adresse du site : Votre nom de domaine(ex : afboxing.com)
3 Cliquez sur "Admin" dans le menu de navigation (en haut à droite) ou allez directement à : Votre nom de domaine(ex : afboxing.com)/admin
4 Connectez-vous avec vos identifiants :
  • Nom d'utilisateur : (fourni par l'administrateur)
  • Mot de passe : (fourni par l'administrateur)
⚠️ Sécurité : Ne partagez jamais vos identifiants. Si vous pensez qu'ils ont été compromis, contactez immédiatement l'administrateur technique(Maxence ADOPO - contact@pixelfounderstudio.fr).
💡 Astuce : Si vous oubliez votre mot de passe, contactez l'administrateur technique (Maxence ADOPO - contact@pixelfounderstudio.fr) qui pourra le réinitialiser.

4. Gestion des actualités

Les actualités apparaissent sur la page d'accueil du site. Vous pouvez en créer, modifier ou supprimer.

➕ Créer une actualité

1 Cliquez sur "Actualités" dans le menu de gauche
2 Cliquez sur le bouton "✨ Créer l'actualité" (en haut à droite)
3 Remplissez le formulaire :
  • Titre * : Le titre de l'actualité (obligatoire)
  • Date * : La date de publication (obligatoire)
  • Résumé * : Un court texte qui apparaît sur la page d'accueil (obligatoire)
  • Description * : Le texte complet de l'actualité (obligatoire)
  • Image : Une photo pour illustrer (optionnel, mais recommandé)
4 Pour ajouter une image :
  • Cliquez sur "Choisir un fichier"
  • Sélectionnez une image sur votre ordinateur
  • L'image s'affichera automatiquement en prévisualisation
5 Cliquez sur "✨ Créer l'actualité" pour sauvegarder
✅ Succès ! Votre actualité est maintenant visible sur le site.

✏️ Modifier une actualité

1 Dans la liste des actualités, cliquez sur l'icône ✏️ Modifier (crayon) à côté de l'actualité
2 Modifiez les champs souhaités
3 Cliquez sur "💾 Modifier" pour sauvegarder

🗑️ Supprimer une actualité

1 Cliquez sur l'icône 🗑️ Supprimer (poubelle) à côté de l'actualité
2 Confirmez la suppression dans la fenêtre qui apparaît
⚠️ Attention : La suppression est définitive. L'actualité ne pourra pas être récupérée.

🔍 Rechercher et filtrer

Vous pouvez utiliser les filtres en haut de la page pour :

  • Rechercher par mot-clé dans le titre ou la description
  • Filtrer par date : sélectionner une période
  • Trier par date (plus récent ou plus ancien)

📦 Actions en masse

Vous pouvez sélectionner plusieurs actualités et les supprimer en une seule fois :

1 Cochez les cases à côté des actualités à supprimer
2 Cliquez sur "Supprimer la sélection"

5. Gestion des palmarès

Les palmarès affichent les résultats de compétitions et les titres remportés par les boxeurs du club.

➕ Créer un palmarès

1 Cliquez sur "Palmarès" dans le menu
2 Cliquez sur "Ajouter un palmarès"
3 Remplissez le formulaire :
  • Titre * : Nom de la compétition (ex: "Championnat de France")
  • Date * : Date de la compétition
  • Lieu : Ville où s'est déroulée la compétition
  • Catégorie * : Type de compétition (Boxe Amateur, Boxe Éducative, etc.)
  • Résultat * : Résultat obtenu (Champion, Vice-champion, etc.)
  • Boxeur / Équipe * : Nom du boxeur ou de l'équipe
  • Détails * : Description détaillée du résultat
4 Cliquez sur "🏆 Créer le palmarès"

6. Gestion de la galerie

La galerie permet d'afficher des photos du club sur le site.

➕ Ajouter une photo

1 Cliquez sur "Galerie" dans le menu
2 Cliquez sur "Ajouter une image"
3 Remplissez le formulaire :
  • Titre * : Nom de la photo (ex: "Entraînement du 15 janvier")
  • Catégorie * : Type de photo (Infrastructure, Coaching, Compétition, etc.)
  • Description : Description optionnelle de la photo
  • Image * : Cliquez sur "Choisir un fichier" et sélectionnez une image
4 L'image s'affiche en prévisualisation une fois sélectionnée
5 Cliquez sur "📸 Ajouter l'image"
💡 Formats acceptés : JPG, PNG, GIF. Taille recommandée : moins de 5 Mo par image.

✏️ Modifier une photo

1 Cliquez sur l'icône ✏️ Modifier sur la photo
2 Modifiez les informations (titre, catégorie, description)
3 Pour changer l'image, sélectionnez un nouveau fichier
4 Cliquez sur "💾 Modifier"

7. Gestion du planning

Le planning affiche les horaires d'entraînement par jour de la semaine.

📅 Modifier le planning

1 Cliquez sur "Planning" dans le menu
2 Vous voyez les jours de la semaine (Lundi, Mardi, etc.)
3 Pour chaque jour, vous pouvez :
  • Ajouter un créneau : Cliquez sur "+" pour ajouter un horaire
  • Modifier un créneau : Changez l'heure, l'activité ou le niveau
  • Supprimer un créneau : Cliquez sur la poubelle 🗑️
4 Pour chaque créneau, remplissez :
  • Heure : Ex: "18h00-19h00"
  • Activité : Ex: "Boxe Éducative", "Boxe Loisir"
  • Niveau : Ex: "Tous niveaux", "Débutant", "Avancé"
5 Une fois toutes vos modifications faites, cliquez sur "💾 Sauvegarder le planning" en haut à droite
✅ Important : N'oubliez pas de sauvegarder ! Vos modifications ne sont pas enregistrées tant que vous n'avez pas cliqué sur "Sauvegarder".

8. Gestion des contacts

Les contacts sont les messages envoyés via le formulaire de contact du site.

📬 Consulter les messages

1 Cliquez sur "Contacts" dans le menu
2 Vous voyez la liste de tous les messages reçus
3 Les messages non lus apparaissent en premier et sont marqués différemment

👁️ Marquer comme lu

1 Cliquez sur un message pour le lire
2 Le message s'ouvre avec toutes les informations (nom, email, téléphone, message)
3 Cliquez sur "Marquer comme lu" pour indiquer que vous avez traité le message

🗑️ Supprimer un message

1 Cliquez sur l'icône 🗑️ Supprimer à côté du message
2 Confirmez la suppression

🔍 Filtrer les messages

Vous pouvez filtrer les messages :

  • Tous : Voir tous les messages
  • Non lus : Voir uniquement les nouveaux messages
  • Lus : Voir uniquement les messages déjà traités

9. Gestion des activités

Les activités sont les différentes pratiques proposées par le club (Boxe Éducative, Boxe Loisir, Handiboxe, etc.).

➕ Créer une activité

1 Cliquez sur "Activités" dans le menu
2 Cliquez sur "Ajouter une activité"
3 Remplissez le formulaire :
  • ID unique * : Un identifiant court (ex: "educative"). Généré automatiquement si vide
  • Catégorie : "Boxe" ou "Socio-éducatif"
  • Titre * : Nom de l'activité (ex: "Boxe Éducative")
  • Eyebrow : Sous-titre court (ex: "8–17 ans • Technique • Valeurs")
  • Description * : Description complète de l'activité
  • Nom pour le planning : Nom utilisé dans le planning (ex: "Boxe Éducative")
4 Dans la section "Métadonnées" :
  • Âge : Public cible (ex: "8–17 ans")
  • Équipement : Équipement nécessaire
  • Clé de prix : Référence au tarif (ex: "boxing.educative")
5 Dans la section "Sections de contenu" :
  • Cliquez sur "Ajouter une section" pour créer des blocs de texte
  • Chaque section peut avoir :
    • Un titre
    • Des paragraphes (un par ligne)
    • Des puces (une par ligne)
6 Cliquez sur "✨ Créer l'activité" (le bouton est désactivé si le titre ou la description manquent)
💡 Astuce : Vous pouvez prévisualiser l'activité avant de la sauvegarder en cliquant sur l'icône 👁️ dans la liste.

🔍 Filtrer par catégorie

En haut de la liste, vous pouvez filtrer les activités :

  • Toutes : Voir toutes les activités
  • Boxe : Voir uniquement les activités de boxe
  • Socio-éducatif : Voir uniquement les activités socio-éducatives

🔄 Activer/Désactiver une activité

Vous pouvez activer ou désactiver une activité sans la supprimer :

1 Cliquez sur le bouton ✓ ou ✗ à côté de l'activité
2 Une activité désactivée n'apparaît plus sur le site public

10. Gestion des tarifs

Les tarifs sont gérés depuis la section "Tarifs" de l'admin.

💡 Note : La gestion des tarifs est actuellement en développement donc pas disponiblen. Pour l'instant, les tarifs sont gérés via la base de données directement.

💰 Structure des tarifs

Les tarifs sont organisés par catégorie :

  • Boxe :
    • Boxe Éducative
    • Boxe Loisir
  • Socio-éducatif :
    • Activité Périscolaire

Chaque tarif contient :

  • Label : Nom du tarif
  • Montant : Prix (peut être "Gratuit" ou un nombre)
  • Période : "an" ou "mois"
  • Note : Informations complémentaires

11. Paramètres du site

Les paramètres permettent de modifier les coordonnées du club et les liens vers les réseaux sociaux. Ces modifications apparaissent immédiatement sur tout le site.

📝 Modifier les paramètres

1 Cliquez sur "Paramètres" dans le menu
2 Vous voyez trois sections :
  • Contact : Adresse, téléphone, email
  • Réseaux sociaux : Facebook, Instagram
  • Site : Nom du club, slogan
3 Modifiez les champs souhaités
4 Cliquez sur "💾 Sauvegarder les paramètres" en haut à droite
✅ Important : Les modifications sont immédiatement visibles sur :
  • La page de contact
  • Le pied de page (footer)
  • Tous les endroits où ces informations apparaissent

📧 Coordonnées de contact

  • Adresse : L'adresse complète du club
  • Téléphone : Numéro de téléphone (format : 06 37 23 26 98)
  • Email : Adresse email de contact

🌐 Réseaux sociaux

  • Facebook : URL complète de la page Facebook (ex: https://www.facebook.com/afboxingclub86)
  • Instagram : URL complète du profil Instagram (ex: https://www.instagram.com/afboxingclub86)
⚠️ Attention : Assurez-vous que les URLs des réseaux sociaux sont complètes (commencent par https://).

12. Historique des actions

L'historique enregistre toutes les actions effectuées dans l'administration (création, modification, suppression).

📜 Consulter l'historique

1 Cliquez sur "Historique" dans le menu
2 Vous voyez la liste de toutes les actions avec :
  • Action : Type d'action (Création, Modification, Suppression, Connexion)
  • Entité : Sur quoi l'action a été faite (Actualité, Palmarès, etc.)
  • Description : Détails de l'action
  • Utilisateur : Qui a fait l'action
  • Date : Quand l'action a été faite

🔍 Filtrer l'historique

Vous pouvez filtrer l'historique par :

  • Recherche : Mot-clé dans la description
  • Entité : Type de contenu (Actualité, Palmarès, etc.)
  • Action : Type d'action (Création, Modification, etc.)
  • Date : Période (du... au...)

🗑️ Vider l'historique

⚠️ Attention : Vider l'historique supprime définitivement tous les enregistrements. Cette action ne peut pas être annulée.

Pour vider l'historique :

1 Cliquez sur "Vider l'historique"
2 Confirmez l'action

La recherche globale permet de trouver rapidement n'importe quel contenu dans l'administration.

🔍 Utiliser la recherche

1 Cliquez sur l'icône de loupe 🔍 en haut à droite (ou utilisez le raccourci clavier)
2 Tapez votre recherche (ex: "Boxe Éducative")
3 Les résultats apparaissent par catégorie :
  • Actualités
  • Palmarès
  • Galerie
  • Contacts
  • Planning
  • Activités

14. Conseils et bonnes pratiques

✅ À faire

  • Sauvegarder régulièrement : N'oubliez pas de cliquer sur "Sauvegarder" après vos modifications
  • Vérifier avant de publier : Relisez vos textes avant de les publier
  • Utiliser des images de qualité : Les photos doivent être nettes et bien cadrées
  • Remplir tous les champs obligatoires : Les champs marqués d'un * sont obligatoires
  • Marquer les messages comme lus : Cela aide à suivre les messages traités
  • Utiliser la prévisualisation : Vérifiez comment votre contenu apparaîtra avant de publier

❌ À éviter

  • Ne pas supprimer sans réfléchir : La suppression est définitive
  • Ne pas utiliser de texte trop long : Les textes trop longs sont difficiles à lire
  • Ne pas oublier de sauvegarder : Vos modifications ne sont pas enregistrées automatiquement
  • Ne pas partager vos identifiants : Gardez vos identifiants secrets
  • Ne pas utiliser d'images trop lourdes : Les images de plus de 5 Mo peuvent être lentes à charger

💡 Astuces

  • Utilisez la recherche : Si vous cherchez quelque chose, utilisez la recherche globale plutôt que de parcourir toutes les pages
  • Filtrez les listes : Utilisez les filtres pour trouver rapidement ce que vous cherchez
  • Vérifiez les messages non lus : Consultez régulièrement les nouveaux messages dans "Contacts"
  • Testez sur le site public : Après avoir publié quelque chose, vérifiez qu'il apparaît bien sur le site

15. Résolution de problèmes

🔐 Problème de connexion

1 Vérifiez que vous utilisez les bons identifiants
2 Vérifiez que vous êtes bien sur la bonne adresse (Votre nom de domaine(ex : afboxing.com)/admin)
3 Si le problème persiste, contactez l'administrateur technique(Maxence ADOPO - contact@pixelfounderstudio.fr)

💾 Erreur lors de la sauvegarde

1 Vérifiez que tous les champs obligatoires (marqués d'un *) sont remplis
2 Vérifiez que les dates sont au bon format (AAAA-MM-JJ)
3 Vérifiez que les URLs sont complètes (commencent par http:// ou https://)
4 Lisez le message d'erreur qui s'affiche - il indique souvent le problème

📸 Problème avec les images

1 Vérifiez que le fichier est bien une image (JPG, PNG, GIF)
2 Vérifiez que l'image n'est pas trop lourde (moins de 5 Mo recommandé)
3 Si l'image ne s'affiche pas, essayez de la compresser ou d'en utiliser une autre

🔄 Les modifications n'apparaissent pas

1 Vérifiez que vous avez bien cliqué sur "Sauvegarder"
2 Actualisez la page du site public (F5 ou Ctrl+R)
3 Videz le cache de votre navigateur si nécessaire

❓ Autres problèmes

Si vous rencontrez un problème non listé ici :

  • Notez le message d'erreur exact qui s'affiche
  • Notez ce que vous étiez en train de faire quand le problème est survenu
  • Contactez l'administrateur technique(Maxence ADOPO - contact@pixelfounderstudio.fr) avec ces informations
📞 Contact technique : En cas de problème technique, contactez l'administrateur (Maxence ADOPO - contact@pixelfounderstudio.fr)du site avec :
  • Le message d'erreur exact
  • La section où le problème est survenu
  • Les étapes pour reproduire le problème
